函数管理需求说明

业务说明


产品说明书

https://ycnfozkwomo9.feishu.cn/docx/AydLdfcbtoLkL8x8Fupcy4W1nAd?from=from_copylink


目标场景和背景说明

1、核心目标是将函数相关业务作为独立应用开发，降低业务耦合度，提升维护与开发效率。


功能说明

【函数管理】用于创建、编辑、删除、查询函数，具体功能包括：

函数基础管理：支持函数的增删改查，配置函数名称、分类、返回值类型等基础信息。

函数代码管理：编写、修改函数的 JavaScript 代码，确保函数有返回值。

函数测试：对函数进行测试，输入参数查看输出结果。

函数查询筛选：通过函数名称、分类等条件查询函数，支持排序与筛选。


字典说明：选自系统配置->字典管理

1. 函数分类

分类名称     说明

数学函数     用于数学计算的函数，如求和、乘法等

文字函数     用于文本处理的函数，如字符串拼接、截取等

逻辑函数     用于逻辑判断的函数，如条件判断、布尔运算等

集合函数     用于集合操作的函数，如数组遍历、筛选等

时间函数     用于时间处理的函数，如日期格式化、时间计算等

对象函数     用于对象操作的函数，如属性获取、对象转换等

转换函数     用于数据类型转换的函数，如字符串转数字、对象转数组等

脱敏函数     用于数据脱敏处理的函数，如敏感信息隐藏等

2. 返回值类型

类型名称     说明

数字     函数返回数值类型

文字     函数返回字符串类型

布尔     函数返回布尔值

对象     函数返回对象类型

数组     函数返回数组类型


权限说明

操作权限：

管理员：可创建、编辑、删除所有函数，修改函数代码和配置。

普通用户：可查看权限内的函数，进行测试操作，无法修改或删除函数。

页面权限：按需分配


数据说明

1. 函数基础数据


字段名称     说明     约束 / 规则

函数名称     函数的显示名称     由英文、数字、下划线组成，唯一标识，必填

函数分类     函数所属分类     从函数分类字典选择，必填

返回值类型     函数返回值的数据类型     从返回值类型字典选择，必填

函数代码     函数的 JavaScript 实现     必须包含返回值，必填

函数说明     函数的用途和使用说明     文本描述，必填

创建时间     函数创建时间     系统自动生成，只读

更新时间     函数最后修改时间     系统自动生成，只读

创建人     创建函数的用户     系统自动关联，只读


交互说明

查询

搜索框：

文本框输入函数名称进行模糊查询，按回车触发查询。

下拉框筛选：

筛选函数分类（数学函数、文字函数等），自动触发查询。

筛选返回值类型（数字、文字等），自动触发查询。

表格头筛选：

支持按【函数名称、函数分类、返回值类型】筛选。

表格头排序：

支持按【创建时间、更新时间、函数名称、返回值类型】排序。


新增 / 编辑函数

必填字段校验：

函数名称、分类、返回值类型、函数代码、函数说明为必填项，未填写时表单无法提交。

函数代码必须包含 return 语句，否则提示错误。

代码编辑：

提供代码编辑器，支持 JavaScript 语法高亮。

自动填充函数模板，如function functionName() { return; }。

测试功能：

输入测试参数，如sum(1, 2, 3, 4)。

点击 “测试” 按钮，调用函数并显示输出结果。


提交交互：

点击 “提交” 按钮后，前端验证所有字段格式和代码正确性。

提交后按钮显示加载状态，发送请求至后端。

成功后关闭窗口，更新函数列表；失败保持窗口，提示原因（如代码错误、名称重复）。


提交

点击 “提交” 按钮后，前端验证所有字段格式，通过后发送请求至后端,按钮加载状态，拿到结果后解除，成功关闭窗口失败保持窗口，告知原因，方便用户修改。


危险操作

删除处理流：弹出确认弹框，提示 “删除后处理流及关联数据无法恢复，是否继续？”，确认后执行删除操作。


其他交互

无特殊情况，按原型开发即可不做任何说明。
