目标场景和背景说明

1、核心目标是将风控设置相关业务作为独立应用开发，降低业务耦合度，提升维护与开发效率。


功能说明

【风控设置管理】主要用于管理 API 访问白名单，具体功能包括：

 白名单开关控制：启用或停用 API 访问白名单功能，控制是否开启 IP 访问控制。主要目的是防攻击和允许外部系统调用决策

 白名单管理：添加、删除、查询 API 访问白名单中的 IP 地址


字典说明：选自系统配置->字典管理

1. 白名单状态

状态     说明

启用     白名单功能开启，仅允许列表内 IP 访问 API

停用     白名单功能关闭，所有 IP 均可访问 API

2. IP 地址格式

格式要求     说明

标准 IP 格式     支持 IPv4 格式，如 192.168.1.1

IP 段格式     支持 CIDR 表示法，如 192.168.1.0/24
